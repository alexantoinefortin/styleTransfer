# styleTransfer
Implementation of Universal Style Transfer via Feature Transforms, `Yi et al. (2017)`, with Keras

#### Auto encoder
* CNN Auto encoder
* Trained on `<insert-dataset-name>`
* Increasing Batch Size on Epoch begin as described in `Smith, Kindermans, Le (2017)`
